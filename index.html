Overview

The RAuth Application is an User Authentication Application built using the MEAN Stack Technologies. The application

  1. Allows a user to register.
  2. Saves the user data in MonboDB, password is encrypted before storing.
  3. Let a returning user log in.
  4. Keep a logged in user’s session alive between page visits.
  5. Provide authorized access to Home and Profile pages, and prevents any unauthorized access to these pages.
  6. Change the User Interface depending on logged in status (e.g. a “login” button or a “my profile” button).
  7. Allow a valid user to restore the password securely.
